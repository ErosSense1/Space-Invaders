# Space-Invaders (Controls)

| Key | Action |  |
| ------------- | ------------- | ------------- |
| Left Arrow | Move to left  | ( < ) |
| Right Arrow | Move to right  | ( > ) | 
| Space | Shoots |

# **can edit controls with:**

```javascript
  player.keys = {
    left: {
      key: "Your Key".
      pressed: false
    },
    right: {
      key: "Your Key",
      pressed: false
    }
  };
// This can be found on class.js-( line:15, column: 9 )//
```
