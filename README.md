# Space-Invaders

controls: 
  left: ArrowLeft
  right: ArrowRight
  
can edit controls with:
  ``player.keys = {
    left: {
      key: "Your Key".
      pressed: false
    },
    right: {
      key: "Your Key",
      pressed: false
    }
  };``
